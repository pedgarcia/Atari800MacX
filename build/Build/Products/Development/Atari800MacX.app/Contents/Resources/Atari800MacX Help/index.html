Atari800MacX Help

Atari800MacX is a Macintosh OSX port of David Firth's Fantastic Atari 800 Emulator.  It is intended to support all of the features of the emulator core, while providing a native Mac Cocoa interface for ease of use.  It leverages the Sound, Video, and Input capabilities (including full Mac OSX joystick/gamepad support) of the libSDL graphics library on the Mac OSX platform.

The emulator requires OS X, and also requires ROM image files from the original Atari systems, which are not provided with the emulator.
