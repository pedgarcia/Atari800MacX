Touch Bar



The Touch Bar on the Atari800MacX emulator allows you to simulate key presses for manu of the speical Atari keys. The Touch Bar has the following buttons:

Power
This performs a Cold boot, or power cycle of the emulated Atari.

Reset, Select, Option, Start
This is the same as pressing these standard function keys on the Atari.

Clear
This is the same as pressing the clear screen key on the Atari.

Inverse
This is the same as pressing the inverse key on the Atari. (This is the Fuji key on a 400/800 or the diagonaly split black and white rectangal on a XL/XE).

Help
This is the same as pressing the Help key on the Atari XL/XE.

Ins/Del/F
This brings up a Touch Bar popover with additional keys:



Ins C, Ins L, Del C, Del L
These correspond to pressing the Insert Char, Insert Line, Delete Char, and Delete Line keys on the Atari.

F1, F2, F3, F4
These correspond to pressing the F1, F2, F3, and F4 kes on the Atari XL/XE.
